# Doodler_wxWidgets-
wxWidgets APPLICATION
This is a sample Doodler Application
Three classes
  - DoodlerMain 
    - initializes the canvas and the tool bar classes
  - DoodlerTool
    - tool bar class that has functionality such as:
      - clearing the screen
      - saving the screen as .bmp
      - loading a .bmp file on to the screen (must be size of screen 500,500)
      - drawing on the screen, square,circle,or line
      - change thickness of line
  - MyCanvas
    - canvas that allows the user ot draw on it
    - draw rectangle, circle,line, image effects
